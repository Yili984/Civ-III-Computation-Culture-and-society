Kirnberger Musical Dice Game

Algorithmic Composition — 379 Trillion Unique Mozart-Style Pieces

UChicago · Civ III: Computation, Culture, and Society · Spring 2026

What Is This?

In the 18th century, composers like Kirnberger and Mozart created Musikalisches Würfelspiel — musical dice games where you roll dice to select pre-composed melodic fragments and assemble them into a unique piece. The result is a system that generates 379,749,833,583,241 distinct compositions from a single table.

This project implements that game as an interactive Python GUI with dice-rolling animation, live progress tracking, full MIDI output with chord accompaniment, and a score viewer. It runs as a desktop app (Python + tkinter). Clone the repo below to run it locally.

How to Play

1. Launch  —  py kirnberger_game_gui.py (or double-click PLAY_GAME.bat)

2. Roll  —  Click Roll Dice — animated result 1–11 selected for the current bar

3. Repeat  —  Roll 14 times total, watch your composition build bar by bar

4. Listen  —  Click Play Music to hear your unique piece via MIDI

5. Save  —  Export as .mid and open in MuseScore, VLC, or any DAW

Features

🎲

Animated Dice Rolling

Clickable dice with animation, random 1–11 result per bar, instant visual feedback.

🎵

Full MIDI Output

Two-track MIDI: melody (right hand) + Alberti bass chord accompaniment (left hand).

📊

Score Viewer

Bar-by-bar breakdown showing harmony labels, notes, and dice roll history.

🎼

Classical Harmony

Correct voice leading across 14 bars with Period I and Period II progressions.

💾

Save & Share

Export compositions with custom filenames. Open in MuseScore, VLC, or any DAW.

🔄

Infinite Replay

Every game produces a statistically unique piece — you will never repeat yourself.
